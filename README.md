# Shop_Pj Итоговое задание 35.6 (HW-07)

## 1. Добавить фичу «похожие товары».

- Shop.API;
- Shop.Admin;
- Базу данных.

## 2. Реализовать функционал создания товара в админке.

Эта доработка затронет:

- Shop.API;
- Shop.Admin.
